# English Learning App

Ứng dụng học tiếng Anh toàn diện với các tính năng:
- Học ngữ pháp
- Luyện phát âm IPA
- Học từ vựng
- Luyện hội thoại

## Tính năng chính

### Ngữ pháp
- Danh sách các điểm ngữ pháp
- Bài học chi tiết với ví dụ
- Bài tập thực hành
- Theo dõi tiến độ học tập

### Phát âm IPA
- Học bảng phiên âm quốc tế
- Luyện phát âm với audio
- Bài tập nhận diện âm
- Đánh giá phát âm

### Từ vựng
- Học từ vựng theo chủ đề
- Bài tập đa dạng
- Luyện nghe và phát âm
- Theo dõi tiến độ học tập

### Hội thoại
- Các tình huống giao tiếp
- Luyện nói và nghe
- Đánh giá phát âm
- Theo dõi tiến độ

## Công nghệ sử dụng

- Flutter
- Firebase Authentication
- Cloud Firestore
- Google Sign-In
- Flutter TTS
- Flutter Sound
- Whisper API

## Cài đặt

1. Clone repository:
```bash
git clone https://github.com/your-username/english_learning_app.git
```

2. Cài đặt dependencies:
```bash
flutter pub get
```

3. Cấu hình Firebase:
- Tạo project trên Firebase Console
- Thêm ứng dụng Android/iOS
- Tải và thêm file cấu hình Firebase

4. Chạy ứng dụng:
```bash
flutter run
```
